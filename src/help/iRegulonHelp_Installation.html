iRegulon Help


Install instructions

1. Place the jar (iRegulon_fat.jar) in the plugin folder of your Cytoscape installation (usely Cytoscape_v2.8.2 > plugins).
2. Start or restart Cytoscape
3. You can find the plugin in the menu: Plugins > iRegulon
